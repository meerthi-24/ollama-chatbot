# Local LLM Chatbot using Ollama & Flask

This is a locally hosted AI chatbot built using:

- Python (Flask)
- Ollama
- Llama 3.2
- HTML/CSS/JS

## Features
- Context-aware chat
- REST API integration
- Local LLM deployment
- Reset chat option

## How to Run
1. Install requirements
2. Run Ollama
3. Run python app.py
